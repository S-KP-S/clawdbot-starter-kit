## QMD Architecture Explained
- QMD is a local RAG retrieval engine with ML models:
  - 300MB embedding model for vectors
  - 640MB reranker model for relevance scoring
  - 1.1GB query expansion model
- WSL memory usage (~13GB) is from these models loaded
- Can shut down WSL freely - models reload on-demand (10-20s cold start)
- Spencer has 64GB RAM - plenty of headroom
